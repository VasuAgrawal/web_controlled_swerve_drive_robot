bool network_module::data_available

Value showing whether or not a new data piece is available.

This is set to true whenever a complete data pair is received, and set to false again whenever the server is in the middle of crafting a pair.

bool network_module::key_received

Whether or not a key has been received.

If both key and value are received, data_available is marked as true and both key_received and value_received are reset.

YunClient network_module::remote

Remote client object, on the controller side.

Currently the client must be on the same local network as the server for the connection to work.

YunServer network_module::server

Server object on the Yun's side.

Currently the Yun creates its own network, so a client needs to be connected to the local network in order to be able to access the server.

bool network_module::value_received

Whether or not a value has been received.

If both key and value are received, data_available is marked as true and both key_received and value_received are reset.
